My Works:

This is an example of a small confectionery web page, the purpose of which is to familiarize the client with the benefits of the product and encourage him to make an application and buy the products.

To create the daribox.com site, the MVC design pattern was used, which allowed separating the site logic from the user interface. To manage the database on the server - DBMS - MySQL was installed.. The basket work is organized with using js, and all data about the purchased product is stored in localStorage on the client side. Site control is carried out from the admin panel, where you can edit prices and product names, moderate user reviews. In this project also implemented interaction with API Liqpay to pay for a certain type of goods. (https://daribox.com/mail). The site is constantly evolving and improving.
