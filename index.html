Примеры работ:

Сайт lending.hostenko.com - это пример небольшой странички кондитерской, цель которой ознакомить клиента с преимуществами товара и побудить оставить заявку на приобретение продукции.

Для создания сайта daribox.com был использован шаблон проектирования MVC, что позволило отделить логику сайта от пользовательского интерфейса. Для управления базой данных на сервере установлена СУБД - MySQL. Работа корзины организована с помощью js, а все данные о приобретенном товаре храняться в localStorage на стороне клиента. Управление сайтом осуществляется из админ панели, где можно редактировать цены и наименования товаров, модерировать оставленные пользователями отзывы. Также в проекте реализовано взаимодействие с API Liqpay для оплаты определенного вида товаров (https://daribox.com/mail). Сайт постоянно развивается и совершенствуется.
